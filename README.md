# TaskFlow - To-Do Web App

A modern and responsive To-Do List web application built using jQuery and localStorage.

## 🚀 Features

* Add, delete, and complete tasks
* Filter tasks (All / Completed / Pending)
* Dark mode toggle
* Persistent storage using localStorage
* Clean and responsive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript (jQuery)

## 📂 Project Structure

* index.html
* style.css
* script.js

## 📌 How to Use

1. Enter a task
2. Click "Add" or press Enter
3. Manage tasks easily

## 🌐 Future Improvements

* Due dates
* Drag and drop
* Backend integration
